========================
1️⃣ Partie Utilisateur
========================

Class: User
- id
- name
- email
- password
- phone
- photo
- city
- address
- description
- status (active | suspended)
- createdAt

Class: Permissions
- id
- name (e.g., "create_product", "accept_order", "view_all_orders")
- description

Class: UserPermissions
- role (CLIENT | PATISSIERE ...)
- permissionId

Class: Client extends User
- walletBalance
- ordersIds[] (String[])
- ratingAverage (Nim)

Class: Patissiere extends User
- bio
- followersCount
- ratingAverage
- completedOrdersCount
- products[] (Product)
- earnings

Class: Livreur extends User
- vehicleType
- ratingAverage
- deliveriesCompleted (le nombre de livraisons que le livreur a terminées avec succès)
- earnings

Class: Admin / Manager extends User
- permissions

Class: SuperAdmin extends Admin
- fullAccess
